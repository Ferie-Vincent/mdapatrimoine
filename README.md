# SCIManager - MDA Patrimoine

Plateforme web de gestion immobiliere et de patrimoine multi-SCI.

## Fonctionnalites

- Gestion multi-SCI avec cloisonnement des donnees
- Gestion des biens immobiliers, locataires et baux
- Suivi des paiements et echeances mensuelles
- Point financier courant (prestations, achats, charges fixes)
- Generation de documents PDF (quittances, recus, attestations)
- Dashboard et analytique
- Exports CSV/Excel
- Application PWA responsive (desktop, tablette, mobile)
